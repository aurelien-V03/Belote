Auteur : Aurelien Vallet
Date : Decembre 2019
Project Name : Belote

*********************

Ce jeu de belote est une version simplifiée du véritable jeu de belote, ici l'utilisateur a accès à un plateau
de belote ou 4 joueur s'affronte, celui-ci peut donc jouer des parties autant de fois qu'il veut.
L'idée du projet etant plus d'appliquer le concept de programmation orientée objet (POO) à un cas précis.
